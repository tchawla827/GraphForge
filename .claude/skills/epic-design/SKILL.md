# Epic Design Skill

Create stunning, cinematic web designs with advanced animations, depth layering, and accessibility built-in.

## How to Use

Provide your design brief and assets, and I'll generate a self-contained HTML file ready to deploy.

### Step 1: Describe Your Vision

Tell me about your project:
- Product type (SaaS, portfolio, product launch, etc.)
- Desired mood/feeling (luxury, playful, futuristic, minimalist, etc.)
- Number of sections you want
- Key message or focus

### Step 2: Provide Assets

Share 3-5 key images/assets:
- PNG with transparency preferred (best results)
- JPEG acceptable
- I'll automatically calculate depth levels and compositional hierarchy

### Step 3: Get Your HTML

I'll deliver a single self-contained HTML file featuring:
- 6+ depth layers minimum
- GSAP animations via CDN
- Built-in accessibility (`prefers-reduced-motion` support)
- Mobile performance optimization
- Inline CSS + JavaScript
- Detailed comments explaining every technique from the 45-technique catalogue

## Activation Triggers

This skill activates automatically when you mention:
- "Make it feel alive" or "Apple-style animation"
- "Sections that overlap" or "product rises between sections"
- "Immersive scrollytelling" or scroll-driven visual effects
- Any cinematic or immersive web design task

## Example Usage

```
/epic-design luxury SaaS product launch with 4 sections
```

Then provide your images and I'll build the design.

## Tips

- **Use transparent PNGs** for best visual results
- **Describe the mood clearly** — "luxury" vs "playful" vs "futuristic" produces very different animations
- **Provide 3-5 images** — the more quality assets, the richer the composition
- **Specify layout** — "hero + features + pricing + CTA" helps me plan the hierarchy
- The agent automatically optimizes for mobile without extra instructions

## Output

Single HTML file with:
- Zero dependencies (all via CDN)
- Copy-paste ready
- Production-optimized
- Fully documented inline
